# ttc
지하철 노선정보를 받아 검색하고 즐겨찾기 할 수 있는 앱

## 사용한 개념 및 기술
- MVVM & AAC
- Kotlin(with Coroutine, Flow)
- Compose
- 클린 아키텍쳐
- UseCase 패턴
- Repository 패턴
- 모듈러
